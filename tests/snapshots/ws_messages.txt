# sonote WebSocket 메시지 타입 스냅샷
# 생성일: 2026-03-16
# 추출: src/server.py WebSocket 핸들러 분석
# 엔드포인트: /ws/transcribe

## 서버 → 클라이언트 (send)
transcript      — 전사 결과 세그먼트
ping            — 연결 유지 핑 (ts: float)
pong            — ping 응답 (ts: float)
error           — 오류 메시지 (message: str)
search_result   — 검색 결과

## 클라이언트 → 서버 (receive)
ping            — 연결 유지 핑
search          — 세션 내 검색 요청
(기타)          — 알 수 없는 타입은 error 응답 반환

## SSE 스트림 (/stream) 이벤트
transcript      — 전사 세그먼트 브로드캐스트
correction      — 텍스트 교정 이벤트
